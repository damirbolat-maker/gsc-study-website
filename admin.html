🐙 GitHub — хранилище данных

          

            Данные сайта хранятся как JSON файлы в GitHub репозитории.

            Нужен Personal Access Token с правами на repo.
          

          

            

            

            

          

          

            
            
          

          

        

        

          
🔐 Как создать токен GitHub

          

            
1. Зайди на github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)

            
2. Нажми Generate new token (classic)

            
3. Выбери срок действия — No expiration

            
4. Отметь repo (полный доступ к репозиторию)

            
5. Нажми Generate token и скопируй — он показывается один раз!
